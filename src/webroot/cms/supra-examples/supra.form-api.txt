Supra.Form class API:

	addInput(configuration)
		Add input to the form.
		Configuration - input configuration object:
			id         - unique field ID
			name       - input name, will be sent as name on save request. Optional
			label      - label text. Optional
			type       - field type. Possible values: "String", "Path", "Template", "Checkbox", "Select", "Text", "Html". Optional, default is "String"
			srcNode    - input node or field (input,label,etc.) node. Optional
			labelNode  - label node, label property will be set to text content of labelNode. Optional
			value      - input value. Optional, default is empty string,
			disabled   - input disabled or not. Optional, if input has disabled attribute, then it will be used
			
	setInput(configuration)
		Alias for addInput
		
	getSerializedValues(key)
		Returns serialize values ready for use in query string
		key - Name of the property, which will be used for key, default is 'name'
		
	getValues(key)
		Returns object with all input values
		key - Name of the property, which will be used for key, default is 'name'
		
	getInputs()
		Returns all input objects in Object, where key matches input ID 
		
	getInput(id)
		Returns input object matching ID
		
	getConfig(id)
		Returns input confguration
		
	setAutoDiscoverInputs(auto)
		During setup search for input,textarea,select elements in form and create input objects
		auto - default is true
		
	setURLLoad(url)
		Set load request URL
	
	setURLSave(url)
		Set save request URL
		
	setURLDelete(url)
		Set delete request URL
		
	save()
		Validate and execute save request if url is set and user is authorized
		
	load()
		Load data and populate form if url is set and user is authorized
		
	delete()
		Execute delete request if url is set, form has ID field and user is authorized to delete record
		

Static methods
	Supra.Form.factoryField(configuration)
		Create input from configuration.


Supra.Input class API
	setValue(value)
		Set input value
		
	getValue()
		Returns input value
	
	setDisabled(disabled)
		Disable or enable input
		
	getDisabled()
		Returns true if input is disabled, otherwise false 
	
	setLabel(label)
		Changes input label, creates it if needed
	
	getLabel()
		Returns label
	
	addClass(classname)
		Add classname to inputs srcNode
		
	removeClass(classname)
		Remove classname from inputs srcNode
		
	hasClass(classname)
		Returns true if inputs srcNode has classname, otherwise false
		
	toggleClass(classname)
		Toggle classname for ininputs srcNode
		
	getAttribute(attr)
		Returns objects attribute value or if it doesn't exist, then input element attribute value
		
	showError(message)
		Show error message
	
	hideError()
		Hide all error messages for input
		
	validate()
		Validate input value
