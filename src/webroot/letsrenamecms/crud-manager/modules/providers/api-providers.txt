Supra.CRUD - namespace for modules

Supra.CRUD.Providers
	Attributes:
		activeProvider
			Active provider ID
		
		configuration
			All provider configuration
	
	Properties:
		providers
			Provider list
	
	Methods:
		getProvider(id)
			Returns provider by ID or null if not found
	
		getActiveProvider()
			Returns active provider
	
		setActiveProvider(id)
			Sets active provider
		

Supra.CRUD.Provider
	Attributes:
		id
			Provider ID
		
		active
			True if provider is active, only one provider can be active at any time
		
		title
			Provider title
		
		delete
			True if it is allowed to delete records
		
		create
			True if it is allowed to create new records
		
		sortable
			True if user can sort record manually
			
		locale
			True if locale selector should be available
			
		recordId
			Record ID for item which is currently beeing edited, null in list mode
	
	Properties:
		fields
			All field configuration
		
		fields_list
			List of fields for list mode
			
		fields_edit
			List of fields for edit mode
	
	Methods:
		getForm()
			Returns form instance
		
		getDataGrid()
			Returns data table instance
		
		getFields()
			Returns all field configuration

		getField(field_id)
			Returns field configuration

		getListFields()
			Returns configuration for all list mode fields

		getEditFields()
			Returns configuration for all edit mode fields 

		getRecordId()
			Returns currently edited record ID
			
		getRecord()
			Returns currently edited record data
